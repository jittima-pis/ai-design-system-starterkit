# Documentation Pages

This folder is for documentation pages (component demos, design tokens showcase, etc.)

## Note

Documentation pages are currently in the main `/app` folder. You can move them here to organize better:

- `/modes` → `/(docs)/modes`
- `/mode-colors` → `/(docs)/mode-colors`
- `/design-tokens` → `/(docs)/design-tokens`
- Component demo pages

## Route Groups

Next.js route groups `(marketplace)` and `(docs)` allow organizing files without affecting URL structure.
